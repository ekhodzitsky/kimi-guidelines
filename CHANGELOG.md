# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.2.0] - 2026-05-04

### Added

- **Mechanized contract verification**: `scripts/check-contracts.py` — real script that verifies Hoare triples, unwrap usage, and SAFETY comments
- **Kani model checker integration**: Proof harnesses in `examples/rust-demo/src/kani_proofs.rs`, CI job, documentation in FORMALISM.md §5
- **Working demo**: `examples/rust-demo/` compiles, passes clippy, has doc tests for all public functions
- **CI verification**: Contract checker + cargo test + clippy + doc on demo project
- **`.cargo/config.toml`** committed to repo (not just generated by install.sh)
- **Real merge example** in `examples/existing-project/AGENTS.md`

### Changed

- **Root AGENTS.md simplified** from 321 lines to ~100 lines — 5 rules with examples, self-contained
- **Rebranding**: "mathematical proof" → "structured contracts" / "correctness by construction"
- **Templates fixed**: `full/` is actually full, `rust-only/` is concise summary
- **README**: honest description, before/after example, no broken curl|bash promise

### Removed

- Swift support (focus on Rust)
- Skills directory (vaporware)
- Excess academic vocabulary from root documents (moved to optional reference docs)

## [1.0.0] - 2026-05-04

### Added

- Initial release with Rust guidelines
- Base rules, GLOSSARY, PIPELINE, SEVERITY, FORMALISM docs
- `languages/rust/AGENTS.md` with Hoare triples and property tests
- `templates/minimal/`, `templates/rust-only/`, `templates/full/`
- `install.sh` with interactive and non-interactive modes
- `CHANGELOG.md`, `LICENSE` (MIT), `.gitignore`
